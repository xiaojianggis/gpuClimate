============
Installation
============


Stable release
--------------

To install gpuclimate, run this command in your terminal:

.. code-block:: console

    $ pip install gpuclimate

This is the preferred method to install gpuclimate, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for gpuclimate can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://gitlab.com/lixiaojianggis/gpuclimate

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://gitlab.com/lixiaojianggis/gpuclimate/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Gitlab repo: https://gitlab.com/lixiaojianggis/gpuclimate
.. _tarball: https://gitlab.com/lixiaojianggis/gpuclimate/tarball/master
